# Day 1 — Cloud Basics
## Cloud Kya Hai?
Internet pe servers available hote hain
Cloud = Computers on the internet 
that store our data, run our apps, 
and connect us with the world
— all on rent!
## Server Kya Hai?
24/7 chalne wala powerful computer
Server = A high performance computer 
on internet that works 24/7

## IaaS — Infrastructure as a Service
Khali kamra milta hai 🏠
Matlab Sirf basic cheezein milti hain
Baaki sab tum khud karo
jisme hame apna platform or software 
khud banana padta hai fir ham apni dukan chalu kar sakte hai

English
IaaS — Infrastructure as a Service
We get only basic things (empty room) 🏠
We have to build our own platform 
and software ourselves.
After that we can start our own shop.

## PaaS — Platform as a Service
Sajaa kamra milta hai 🛋️
Matlab hame kamra bhi mila or usme ferniture bhi
aab hame isme apna software banana hai matlab code likhna hai 
fir ham apni dukan chalu kr sakte hai

English
PaaS — Platform as a Service
We get a furnished room 🛋️
It means we get the room 
along with the furniture too.
Now we just have to build 
our own software (write code).
After that we can start our own shop.

## SaaS — Software as a Service
Chalu dukaan milti hai 🏪
Matlab hame kamra bhi mila
furniture bhi hai 
or dukan ka sara saman bhi hai 
hame bas chalana hai

English
SaaS — Software as a Service
We get a ready running shop 🏪
It means we get the room,
the furniture is already there,
and all the shop items are also ready.
We just have to run it.
