# Day 6 — Linux Basics

## Linux Kya Hai?
Free and open source OS
Amazon, Google, Facebook use karte hain
English 
Linux is open source operating system
Amazon, Google, Facebook, are use it.

## Linux Commands:
pwd   = Print Working Directory
ls    = List
cat   = Concatenate
mkdir = Make Directory
cd    = Change Directory
touch = Touch (nayi file banao)
echo  = Echo (file mein likho)

## Practice Ki:
✅ Folder banaya (mkdir)
✅ File banai (touch)
✅ File mein likha (echo)
✅ File padhi (cat)
