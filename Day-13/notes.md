# Day 13 — CI/CD Pipeline

## CI/CD Kya Hai?

CI = Continuous Integration
     Continuously check code 
     for errors

CD = Continuous Deployment
     After tests pass, 
     automatically deploy

## Bina CI/CD:
Manual test → Manual deploy
(Slow + Error-prone)

## CI/CD Ke Saath:
Push code → Auto test → Auto deploy
(Fast + Reliable)

## GitHub Actions
Tool that creates CI/CD 
pipelines in GitHub

Workflow file location:
.github/workflows/main.yml

## Pipeline Kya Hai?
Sequence of steps that 
work one by one
Example: Code → Test → 
         Build → Deploy

## Seekha:
CI/CD pipeline kya hota hai
aur companies mein kaise 
automation hoti hai! 🔄
