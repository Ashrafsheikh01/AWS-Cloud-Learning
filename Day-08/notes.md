# Day 8 — Networking Basics

## IP Address
Address of computer on internet
Contains 4 numbers
Separated by dots
Each number 0-255

## DNS — Domain Name System
Converts domain name to IP address
google.com = 142.250.77.46

## HTTP vs HTTPS
HTTP  = HyperText Transfer Protocol
        Open + Unsecure ❌
HTTPS = HyperText Transfer Protocol Secure
        Encrypted + Secure ✅

## Ports
Different doors of a computer
Port 80  = HTTP
Port 443 = HTTPS
Port 22  = SSH
Port 3306 = MySQL
