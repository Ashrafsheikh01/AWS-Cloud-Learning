# Day 7 — Linux Advanced Commands

## Nayi Commands:

cp    = Copy
        File copy karo
        cp file.txt folder/

mv    = Move
        File move karo
        mv file.txt folder/

rm    = Remove
        File delete karo
        rm file.txt
        ⚠️ Wapas nahi aati!

chmod = Change Mode
        Permissions change karo
        chmod 777 file.txt
        7 = Read + Write + Execute
