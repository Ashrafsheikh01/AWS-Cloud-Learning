# Day 4 — EC2 Server Launch

## Aaj Kya Kiya?
✅ EC2 Console khola
✅ Launch Instance click kiya
✅ Name: My-First-Server
✅ OS: Amazon Linux
✅ Instance Type: t3.micro (Free)
✅ Key Pair: my-first-key.pem
✅ Network: SSH Anywhere
✅ Server Successfully Launched!
✅ Server Stop Kiya

## EC2 Launch Steps:
1. EC2 Console kholo
2. Launch Instance click karo
3. Name do
4. OS select karo (Amazon Linux)
5. Instance type: t3.micro
6. Key pair banao
7. Network settings: SSH Anywhere
8. Launch Instance!

## Important:
Server use karo = Start karo
Kaam khatam = Stop karo
Free tier hours bachenge! ✅
