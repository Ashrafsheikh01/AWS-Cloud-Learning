# Day 3 — EC2 Introduction

## EC2 Kya Hai?
EC2 = Elastic Compute Cloud

## Definition:
EC2 is a virtual server on internet
Like my own computer but:
✅ Works 24/7
✅ Available on internet
✅ Controlled by me
✅ Located in Mumbai region

## EC2 Full Form:
E = Elastic (badha chhota kar sakte ho)
C = Compute (computing power milti hai)
2 = Cloud (internet pe hai)

## AWS Console:
console.aws.amazon.com/ec2
